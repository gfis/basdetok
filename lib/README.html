For a build of the application with ant dist, the lib directory should be populated with the following libraries (perhaps with higher versions): 
from https://github.com/gfis/common
  git clone common ; cd common ; ant dist ; cp -v lib/common-core.jar your-lib
from https://commons.apache.org/proper/commons-fileupload/download_fileupload.cgi
  commons-fileupload-1.3.2.jar
from https://commons.apache.org/proper/commons-io/download_io.cgi
  commons-io-2.5.jar
from https://logging.apache.org/log4j/2.x/download.html:
  log4j-api-2.17.1.jar 
  log4j-core-2.17.1.jar
